Welcome to PyVogelSim's documentation!
======================================

**PyVogelSim** is a Python port of Jonathan Vogel's Optimum-Mindstorm lap simulator. If is built
with modularity and consistency in mind, and offers a *performative* and *intuitive* interface for
simulating FSAE laptimes.

.. note::

   This project is under active development.
